# 制約
- 16:9 の任意の解像度のみ対応
- UIサイズが「最高」のみ対応
- 水墨侠客団コレクトのみ対応

# 使い方
|ショートカットキー|動作|
|:--|:--|
|Z + マウスの戻ボタン + 左クリック|コレクトをコレクトボックスからインベントリに取り出す|
|X + マウスの戻ボタン + 左クリック|コレクトを「倉庫からインベントリ」または「インベントリから倉庫」に移動する|
|X + マウスの戻ボタン + 右クリック|コレクトをインベントリからコレクトボックスに入れる|
